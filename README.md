# 🗞️ Web Scrapping - Portal de Notícias

Este projeto faz parte de um desafio prático de desenvolvimento web e extração de dados utilizando Python. O objetivo é construir um portal de notícias funcional, estilizado e, em seguida, realizar a coleta automatizada de informações para análise de dados.

---

## 🚀 Objetivo do Projeto

O **Grupo 2** é responsável por desenvolver o **Portal de Notícias**. O fluxo de trabalho consiste em:
1.  **Ajustar a estrutura HTML:** Corrigir as lacunas no arquivo `index.html`.
2.  **Layout Visual:** Criar o design do portal utilizando CSS no arquivo `css/style.css`.
3.  **Coleta de Dados:** Desenvolver um script Python para extrair as notícias, categorias e links.
4.  **Análise de Dados:** Gerar insights a partir dos dados coletados.

---

## 🛠️ Tecnologias Utilizadas

*   **Frontend:** HTML5, CSS3.
*   **Backend / Scrapping:** Python 3, BeautifulSoup4.
*   **Análise de Dados:** Python (Pandas/Matplotlib opcional).
